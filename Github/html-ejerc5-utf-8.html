¿Qué pasaría si eliminas la etiqueta "meta charset=utf-8"?

La etiqueta "meta charset=utf-8" sirve para indicarle a la web que tipo de alfabeto tiene que emplear, si la eliminas no podría identificar el lenguaje y no podría usar caracteres como la ñ o reconocer las tildes entre otros.
